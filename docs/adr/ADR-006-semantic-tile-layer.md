# ADR-006: Introduce a Semantic Tile Layer

> **Status:** Accepted
>
> **Date:** 2026-07-19

---

# Context

Gameplay should not depend directly on visual tiles.

---

# Decision

Introduce a semantic layer describing gameplay meaning independently from visuals.

---

# Consequences

Benefits:

- decouples gameplay from art
- simplifies procedural generation
- improves room analysis
- allows changing visuals without changing gameplay
